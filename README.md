# 🛒 Ecommerce Website

This is a simple ecommerce website built using Flask.

## 🚀 Features
- Home Page
- WhatsApp Chat Button
- Clean UI
- Flask Backend

## 🛠️ Technologies Used
- Python (Flask)
- HTML
- CSS

## ▶️ How to Run

1. Install Flask:
   pip install flask

2. Run app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
